# Daily Data Enrichment Pipeline (n8n)

The key differentiator of our AI News Search Assistant is the **Daily Data Enrichment Pipeline**. While the base dataset contains ~21,000 articles from May 10–15, 2026, our system continuously stays up-to-date and adds deep AI analysis to fresh news.

## Architecture

The pipeline is built using **n8n** (a workflow automation tool) and runs automatically every day at 09:00.

### Workflow Steps:

1. **Cron Trigger:**
   - Triggers the workflow daily at 09:00 AM.

2. **Google Alerts RSS Fetch:**
   - Pulls the latest news articles from Google Alerts RSS feeds.
   - Configured keywords: `bank`, `maliyyə`, `iqtisadiyyat`, `SOCAR`, `Azərbaycan`, etc.

3. **GPT-4o Analysis (Enrichment):**
   - Each fetched article is passed to OpenAI's GPT-4o model.
   - The model performs three tasks:
     1. **Relevance Check:** Determines if the article is relevant to the banking/economy sector.
     2. **Sentiment Analysis:** Classifies the sentiment as `pozitiv` (positive), `neytral` (neutral), or `riskli` (risky/negative).
     3. **Summarization:** Generates a concise summary in Azerbaijani (`summary_az`).

4. **Embedding Generation:**
   - Relevant articles are passed to the `text-embedding-3-small` model to generate a 1536-dimensional vector.

5. **Supabase Upsert:**
   - The enriched article, along with its embedding, sentiment, and summary, is inserted into the Supabase `articles` table.
   - The `is_enriched` flag is set to `true`.

6. **Daily Digest Email:**
   - The workflow aggregates the enriched articles and sends a formatted daily digest email to stakeholders using Microsoft Outlook OAuth integration.

## Supabase Schema

The `articles` table is designed to support both the base dataset and the enriched articles:

```sql
create table articles (
    id bigint primary key generated always as identity,
    url text unique not null,
    title text,
    content text,
    category text,
    source text,
    created_at timestamp with time zone,
    
    -- AI fields
    embedding vector(1536),
    is_enriched boolean default false,
    sentiment text,      -- 'pozitiv', 'neytral', 'riskli'
    summary_az text      -- Short Azerbaijani summary generated by GPT-4o
);
```

## How It Appears in the Assistant

When users search via the Telegram Bot or Web Interface:
- Enriched articles are marked with a ✨ **[Enriched]** tag.
- The sentiment is displayed using badges: 🟢 Positive, 🔵 Neutral, 🔴 Risky.
- The concise Azerbaijani summary is shown directly in the search results, saving analysts time.
